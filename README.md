# Tool-alignment-with-image-processing
Este repositório contém o estudo da aplicação de visão computacional para alinhamento de ferramental conectado a um motor de passo. O objetivo é analisar a peça/gabarito que deve-se alinhar a uma composição fixa utilizando somente a leitura da câmera, sem sensores extras.

O projeto surgiu da dificuldade de desenvolvimento de uma máquina neddle winding que necessitava de alinhamento do estator para realizar sua rotina, onde devido a mudanças de diferentes configurações de estatores não era possível implementar sensores de detecção, sendo a visão computacional uma solução eficiente.
